What We Do

The Whitman Coding Club welcomes people of all skill levels! We will meet you at your skill level.

    Beginners
  • Do lessons starting from block coding
  • Learn text based coding
  • Create a website or game
    Intermediate and Advanced Coders
  • Practice text based coding with contests
  • Improve coding skills through lessons
  • Get help on current or past projects

Each month we release a new challenge for our members! At the last meeting of each month, we present our projects and vote on a winner.
